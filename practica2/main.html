Árboles de decisiones¶

La tcnica de aprendizaje automtico conocida como rbol de decisiones emplea una estructura en forma de rbol para tomar decisiones como su propio nombre indica. Los nodos intermedios representan soluciones y los nodos finales proporcionan la prediccin deseada. En cada nodo, se toma una decisin basada en un atributo asignado, utilizando la entropia para determinar los atributos de decisin ms precisos.
En problemas de mayor complejidad se puede emplear el Random Forest Classifier que, como su nombre indica, emplea varios arboles de decisin. Este mtodo es ms preciso por norma general que el rbol de decisin normal, pero tambin ms lento.

A continuacin, mediante la funcion GridSearchCV, se ha realizado una bsqueda de los mejores parmetros para cada uno de los dos modelos

A continuacin, mediante la funcion GridSearchCV, se ha realizado una bsqueda de los mejores parmetros para el modelo de Random Forest. En este caso se genera un diccionario parametros con los parmetros que se quieren probar. A continuacin, se crea un objeto grid de la clase GridSearchCV. Este objeto recibe como parmetros el modelo RandomForestClassifier, el diccionario de parmetros parametros. Para evaluar nuestro modelo, usaremos una validacin cruzada de 5 veces con la precisin, ya que es la mtrica que usa Kaggle la tabla de clasificacin.

Perceptrón¶

Un perceptrn es un modelo matemtico basado en la estructura y funcin simplificadas de una neurona biolgica. Los valores de entrada se utilizan para alimentar el modelo, el cual se ejecuta y produce una salida. Si la salida es igual a la salida deseada, el modelo est funcionando correctamente y no se hacen cambios en los pesos. Si la salida no es la deseada, se ajustan los pesos para reducir el error hasta que el modelo alcance una solucin con un error mnimo.
